# lia — Website UI kit

A pixel-spirit recreation of a personal portfolio site for "lia", built
against the Lia Design System foundations.

## Files

- `index.html` — interactive prototype
- `Nav.jsx` — top navigation with brand wordmark
- `Hero.jsx` — large display hero with lede
- `WorkIndex.jsx` — selected work, list + grid views
- `ProjectCard.jsx` — image + caption project card
- `About.jsx` — biographical column
- `Contact.jsx` — short contact form
- `Footer.jsx` — minimal footer with marks

## Notes

These are recreations of common portfolio-site patterns; not copied from
any specific source. They follow `colors_and_type.css` strictly and use
Lucide icons via CDN where pure typographic marks aren't enough.
